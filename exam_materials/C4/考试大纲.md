# C4 考试大纲

本文档基于《阿里云大模型ACP考试大纲-V3.pdf》，针对《大模型ACP认证教程》中的“C4_交付上线”模块，提取并梳理了对应的核心知识框架、考点明细及考核要求。

## 一、模块整体知识框架与考核权重

在阿里云大模型ACP认证考试中，“C4_交付上线”模块主要对应原大纲中的两大板块，合计在考试中占据 **32%** 的极高权重：
1. **大模型微调（占比 16%）**：对应教程中的模型蒸馏与微调实践。
2. **生产环境应用实践（占比 16%）**：对应教程中的模型部署、生产实践及安全合规。

该模块的核心知识框架包括：模型微调与蒸馏原理、模型在各类云服务环境下的部署方案、大模型内容安全与合规保障。

## 三、备考建议与教程映射总结

为了高效备考 C4 模块（占32%高比重），建议将大纲考点与教程文件进行以下关联学习：

1. **微调与蒸馏链路**：在学习 `4_1` 时，务必理清“微调”的四大要素：**作用、前提、步骤、算法**。动手走通一遍数据集构建 -> 配置参数 -> 启动微调 -> 效果评测的完整闭环。
2. **重点掌握 vLLM**：大纲中明确提到“**掌握**如何使用 vLLM 进行大模型的部署操作”，这是一个高频考点。在学习 `4_2` 和 `4_3` 时，需要熟悉 vLLM 的基本命令、适用场景及性能优势。
3. **百炼与云服务结合**：理解模型在本地部署与在阿里云服务（ECS/PAI/FC）部署的区别，特别是结合函数计算（FC）快速发布AI助手的方案。
4. **安全合规不可忽视**：大模型生成内容的不可控性带来了合规风险，考试中关于**内容安全检查手段**和**云服务安全**的选择题较多，学习 `4_4` 时要重点记忆常见的安全合规检测类型及阿里云的安全解决方案。
