# C2 考试大纲

本文档基于《阿里云大模型ACP考试大纲-V3.pdf》，针对《大模型ACP认证教程》中的“C2_构造问答系统”模块，提取并梳理了对应的核心知识框架、考点明细及考核要求。

## 一、模块整体知识框架与考核权重

在阿里云大模型ACP认证考试中，“C2_构造问答系统”模块涉及的核心知识点在原大纲中占据了 **52%** 的总权重，是考试的重中之重。主要包含以下三大模块：

1. **大模型应用开发（占比 17%）**：对应教程中大模型基础调用与简单答疑机器人的构建。
2. **大模型提示词工程（占比 15%）**：对应教程中通过优化提示词改善答疑质量。
3. **大模型检索增强（占比 20%）**：对应教程中知识库扩展（RAG）、自动化评测及RAG应用的持续优化。

## 三、备考建议与教程映射总结

为了高效备考，建议将大纲考点与教程文件进行以下关联学习：

1. **基础调用与参数控制**：在学习 `2_1_用大模型构建新人答疑机器人.ipynb` 时，重点关注 `temperature`、`top_p` 的调节效果，以及如何传递多轮对话历史。
2. **RAG全链路**：学习 `2_2` 和 `2_5` 时，必须理清从**解析 -> 切片 -> 召回 -> 重排 -> 生成**的完整链路，掌握诸如“句子窗口检索”、“表格增强”等具体优化手段。
3. **Prompt工程**：在 `2_3` 中，务必掌握提示词四大要素和分隔符规范，并能在意图分类等实战中写出合格的系统提示词。
4. **评测体系**：对于 `2_4`，重点了解 RAGAS 评测体系的核心指标（如准确性、相关性），明白如何量化地衡量一个问答系统的好坏。

> **注**：本模块（C2）占整个ACP考试超过一半的权重，建议学员在阿里云百炼平台上多做实操，通过动手编写代码和提示词来加深对概念的理解。
