# C3 考试大纲

本文档基于《阿里云大模型ACP考试大纲-V3.pdf》，针对《大模型ACP认证教程》中的“C3_构建Agent系统”模块，提取并梳理了对应的核心知识框架、考点明细及考核要求。

## 一、模块整体知识框架与考核权重

在阿里云大模型ACP认证考试中，“C3_构建Agent系统”模块对应的主要大纲部分为**“多Agent及多模态应用”**，在考试中占据 **16%** 的权重。

该模块的核心知识框架主要包括：
1. **Agent（智能体）基础与运行机制**：对应教程中Agent基础、工具调用、规划与执行。
2. **多Agent与复杂AI应用构建**：对应教程中多Agent协作、Memory（记忆）、Skill（技能）等高级机制。
3. **多模态与行业应用实践**：大模型在多模态内容生成及具体行业的实际应用体验。

## 三、备考建议与教程映射总结

为了高效备考 C3 模块，建议将大纲考点与教程文件进行以下关联学习：

1. **吃透 Agent 核心机制**：在学习 `3_1` 到 `3_5` 时，必须深刻理解 Agent 是如何通过“感知 -> 思考 -> 行动”闭环工作的，特别是**工具调用（Tools）**和**任务规划（Planning）**的底层逻辑。
2. **多模态与多Agent实操**：考试大纲明确要求“掌握用生成多模态内容”，在学习 `3_3` 和阿里云百炼平台的 Assistant API 时，建议多动手构建几个包含多模态能力的智能体（如图像理解、语音交互）。
3. **关注百炼 Assistant API**：原大纲多次提及“基于百炼 Assistant API 构建智能体”，在实际备考中，应将理论与百炼平台的控制台实操紧密结合，熟悉界面配置与API调用的双重方式。
4. **行业解决方案视野**：除了底层代码，考试也会考察大模型在实际行业（如医疗、教育、娱乐）中的应用形态，学员应适当拓展视野，了解常见业务场景下的解决方案。
