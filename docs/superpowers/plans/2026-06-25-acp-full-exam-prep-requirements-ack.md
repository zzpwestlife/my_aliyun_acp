# 需求确认书：阿里云大模型ACP考试备考三件套

## Agent 理解版功能描述
基于考纲V3 + 教程C1-C5，产出三份独立 Markdown 交付物：知识点梳理手册、考点清单、模拟考试卷（含答案与逐题考点解析），覆盖官方 6 大模块 100% 核心考点。

## 边界条件
- 模拟卷严格贴合官方真实结构：单选 50（各1分）+ 多选 25（各2分）= 100分 / 120分钟。
- 6 模块占比：17/15/20/16/16/16（应用开发/提示词/检索增强/微调/多Agent多模态/生产实践）。
- 命题前先定答案分布蓝图：单选 A/B/C/D≈13/12/13/12；多选 2/3/4 选组合全覆盖（9/11/5）。
- 干扰项围绕认知误区/易混概念，强迷惑性；题干场景化（操作/方案/排错）。

## 已确认假设
1. 官方无案例分析题 → 严格贴合，仅单选+多选两类。（用户确认）
2. 只出 1 套 75 题标准卷，不出扩展题库。（用户确认）
3. 三份全新独立文件，不改动现有材料。（用户确认）
4. 教材→模块映射、权重(高中低)、形式(识记/理解/应用) 由 Agent 依考纲推导。

## Out-of-scope
- 案例分析题独立题型；扩展题库；交互式网页/打印组件；改动现有题库文件。
