# C2 题目质量升级设计

## 背景

当前 `exam_materials/C2` 已完成目录整理与答案解析结构统一，但题目质量仍存在明显短板：

- 题干偏直白，部分题目更像知识点复述而不是正式考试题。
- 干扰项强度不均，有些选项可以一眼排除，区分度不足。
- `考试大纲.md`、`模拟题题目.md`、`模拟题答案解析.md` 三者虽然已能对应，但尚未形成“考纲分层 -> 命题难度 -> 解析复盘”的闭环。

本次优化以 `C2` 作为试点，目标不是简单加题，而是建立一套后续可平移到 `C3/C4` 的题目质量标准。

## 范围

本次只处理以下三个文件：

- `exam_materials/C2/考试大纲.md`
- `exam_materials/C2/模拟题题目.md`
- `exam_materials/C2/模拟题答案解析.md`

不修改 `C3`、`C4` 的题库内容，不调整 `common/` 下的通用资料，不处理 PDF 或旧版归档。

## 目标

### 1. 建立难度分层

将 `C2` 重构为一套 `18` 题的标准样板，形成清晰的基础题、中档题、拔高题梯度：

- 单选题 `10` 题：基础 `4`、中档 `4`、拔高 `2`
- 多选题 `6` 题：基础 `2`、中档 `3`、拔高 `1`
- 场景题 `2` 题：一题偏“诊断与优化顺序”，一题偏“评测与迭代决策”

### 2. 提高题目区分度

通过重写题干和干扰项，让题目更接近正式考试风格：

- 基础题考主干概念、基础流程和参数作用，但不能做成送分题。
- 中档题引入约束条件、小场景或方法比较，重点考方法选型与优化顺序。
- 拔高题只占少量，用于拉开区分度，但必须严格基于课程内容，不出偏题怪题。

### 3. 打通考纲、题目与解析

三份文件要形成一套可验证链路：

- `考试大纲.md` 负责定义知识模块、考点分层、考核层级和命题方向。
- `模拟题题目.md` 负责把这些命题方向转成带梯度的试题。
- `模拟题答案解析.md` 负责解释正确项判断依据、主要错项错误原因，并回连到知识点与考纲映射。

## 设计原则

### 命题原则

- 不允许大量“知识点原句改写题”。
- 不允许干扰项明显异维度，导致考生可以仅靠常识排除。
- 不允许答案分布过度集中在同一字母或固定组合。
- 不允许为了拔高而脱离课程边界。

### 结构原则

- `考试大纲.md` 必须明确哪些考点适合出基础题，哪些适合中档题或拔高题。
- `模拟题题目.md` 必须能看出梯度，但不直接在题面暴露答案线索。
- `模拟题答案解析.md` 必须保留 `标准答案`、`解析`、`知识点`、`课程映射/考纲映射` 等复盘必需字段。

### 复盘原则

- 解析不能只解释“为什么正确”，还应说明关键错项“为什么错”。
- 场景题答案应有明确作答框架，而不是开放散文式描述。
- 整套题应服务于备考训练，而不是只做内容展示。

## 文件级设计

### 一、`考试大纲.md`

重构为“知识模块 -> 考点 -> 考核层级 -> 命题难度映射”的结构，至少覆盖以下内容：

- 模块划分：大模型应用开发、提示词工程、RAG 基础、RAG 优化、RAG 自动化评测
- 每个考点的考核层级：了解 / 熟悉 / 掌握 / 应用
- 每个考点适配的命题难度：基础 / 中档 / 拔高
- 每个模块的出题重点与易错方向

输出结果要让命题不再依赖主观感觉，而能直接从考纲映射到试题设计。

### 二、`模拟题题目.md`

重写为 `18` 题标准样板，结构保持清晰：

- 考试说明
- 一、单选题
- 二、多选题
- 三、场景题
- 提交答案区

题目层面重点调整：

- 基础题保留必要覆盖，但增强干扰项强度。
- 中档题强化条件限制、方法区分和场景判断。
- 拔高题强调综合判断与优先级，而非冷知识。
- 场景题改为可评分、可对照标准答案的格式。

### 三、`模拟题答案解析.md`

与新题一一对应同步重写，保留当前已统一好的文档骨架：

- 使用说明
- 标准答案速查表
- 单选题答案解析
- 多选题答案解析
- 场景题参考答案
- 考点覆盖与复习建议

解析质量要求：

- 每题给出 `标准答案`
- 明确说明 `解析`
- 补齐 `知识点`
- 回连 `课程映射` 与 `考纲映射`
- 对中档题和拔高题增加关键错项分析

## 验收标准

本次优化完成后，应满足以下验收条件：

- `C2` 的核心考点都能在题库中找到明确落点。
- `18` 题内部能看出基础、中档、拔高三层难度。
- 至少一半以上客观题带有条件、场景或比较关系，不再只是概念摘抄题。
- 单选题答案分布不过度集中，多选题组合不过度重复。
- 每道题在解析中都能说明判断依据，高质量题还能解释主要错项。
- 整套材料可以作为后续升级 `C3/C4` 的质量样板。

## 非目标

以下事项不在本次范围内：

- 不同步升级 `C3`、`C4`
- 不新增新的模块目录
- 不重做 `common/` 下的综合模拟卷
- 不处理打印版、PDF 导出或网页化展示

## 风险与控制

### 风险 1：题量增加后可完成性下降

控制方式：

- 保持总题量在 `18` 题，不做过长扩展
- 通过梯度设计保证首次刷题仍能完成

### 风险 2：拔高题变成偏题

控制方式：

- 所有拔高题必须来源于现有课程与考纲边界
- 重点考综合判断，不考超纲冷知识

### 风险 3：题目提升了，但大纲与解析没同步

控制方式：

- 三份文件联动修改
- 验收时以“考纲 -> 题目 -> 解析”完整链路核对

## 后续路径

如果 `C2` 试点效果达到预期，则下一阶段可沿同一标准继续处理：

- `exam_materials/C3`
- `exam_materials/C4`

届时优先复用本次形成的：

- 难度梯度定义
- 干扰项质量标准
- 题目与解析联动模板
- 验收标准
