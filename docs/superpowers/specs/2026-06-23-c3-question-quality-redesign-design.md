# C3 题目质量重做设计

## 背景

当前 `exam_materials/C3/模拟题题目.md` 与 `模拟题答案解析.md` 已经完成一轮扩题，但用户实测后给出两个明确反馈：

- 干扰项太弱，很多选项可以靠常识一眼排除
- 答案分布过于机械，单选大量集中在 `B`，多选大量集中在 `ABC`

这说明当前 `C3` 的主要问题已经不是“题量不够”，而是“命题质量不足”。如果继续在现有题面上小修小补，仍然会保留明显的泄题模式，无法达到真实考试应有的区分度。

本次设计目标不是重新定义 `C3` 的知识范围，而是在保留当前文件结构、题号范围和核心考点覆盖的前提下，对 `C3` 做一轮中等力度的题库重做，使其从“概念识别题集”提升为“需要理解边界、条件和工程判断才能做对”的高质量客观题卷。

## 范围

本次只处理以下两个文件：

- `exam_materials/C3/模拟题题目.md`
- `exam_materials/C3/模拟题答案解析.md`

本次不处理：

- `exam_materials/C3/考试大纲.md`
- `C2`、`C4` 的任何文件
- `common/` 下的综合资料
- 新增 PDF、打印稿或其他导出物

## 核心决策

### 1. 采用“中修重做”而不是小修或整套重写

本轮采用中修方案：

- 保留现有题号 `1-26`
- 保留当前知识点覆盖框架
- 系统重写大部分干扰项
- 对泄题明显或判断路径过直的题目，重写题干
- 同步重写对应解析

不采用“小修”的原因：

- 仅修个别选项无法真正打破答案分布模式
- 一些题目本身的问法已经过于直给，单改选项提升有限

不采用“大修”的原因：

- 用户本轮要求是继续重做 `C3`，但没有要求整套推翻重建
- 大修会显著扩大改动面，增加解析重写成本和回归风险

### 2. 保留题号与结构，重做命题质量

本轮不改 `C3` 的整体组织方式：

- 单选题 `16`
- 多选题 `10`
- 总题数 `26`

保留以下稳定约束：

- 题号不变
- 章节结构不变
- 提交答案区与统计区结构不变

重做重点放在：

- 题干约束条件
- 干扰项质量
- 正确答案分布
- 解析中的错项分析质量

### 3. 用“同维度竞争”替代“常识排除”

所有重写后的题目都应遵守一个核心命题原则：

- 错误选项不能错得太明显
- 错误选项必须和正确选项处于同一问题维度
- 错误选项要错在边界、前提、优先级、适用条件或工程代价

例如：

- 不能用“明显离题”的选项凑数
- 不能把“完全不相关的能力”放进来当干扰项
- 应更多使用“看起来也合理，但不符合当前约束”的竞争项

## 设计目标

### 1. 提升真实考试感

重做后的 `C3` 不应再像“看关键词选定义”，而应更像真实考试：

- 给定场景和限制，判断哪种机制更适合
- 比较两个都合理的方案，选更优先或更完整的一个
- 判断某种架构、模式或工具策略的边界

### 2. 打散答案分布，消除模式泄题

本轮明确把“答案分布自然化”作为设计目标之一。

单选题要求：

- 不允许大量集中在同一选项
- 最终分布可以不完全平均，但必须自然
- 若连续多题答案相同，需要有明确题面支撑，而不是命题习惯导致

多选题要求：

- 不能继续大面积集中在 `ABC`
- 应自然引入 `ABD`、`ACD`、`BCD`、`ABCD` 等组合
- 正确项组合变化必须由题面逻辑驱动，而不是机械打散

### 3. 提高干扰项强度

重做后的干扰项应更多来自以下几类“高质量错法”：

- 机制本身合理，但不适合当前场景
- 顺序看似对，但优先级不对
- 方案能力更强，但工程代价或风险不匹配
- 局部正确，但不能解决题干里的核心矛盾

### 4. 保持知识点覆盖不缩水

虽然重做题面，但不能因为追求难度而丢掉原本 `C3` 应覆盖的主干能力：

- 工具调用与结构化输出
- 工作流、规划与反思
- 多 Agent 协作模式
- 记忆管理
- Skill 设计与沉淀
- 评测驱动开发
- Qwen Code 工作方式

## 文件级设计

### 一、`模拟题题目.md`

保留当前四段结构：

- 单选题
- 多选题
- 提交答案区
- 题型统计

重做方法如下：

- `1-12`：保留知识点方向，但重写明显直给的题干与干扰项
- `13-20`：重点打散多选答案组合，并提高错项竞争性
- `21-26`：保留补充题定位，但把新增题拉到与前半卷同一难度标准

优先重做的题型特征：

- 一眼能排除两个以上选项的题
- 题干几乎没有约束条件的题
- 正确项明显比其他选项“更完整、更先进、更像标准答案”的题
- 多选中只要理解主题就能直接猜 `ABC` 的题

### 二、`模拟题答案解析.md`

解析稿继续保留现有结构：

- 使用说明
- 标准答案速查表
- 单选题答案解析
- 多选题答案解析
- 考点覆盖与复习建议

同步重写要求：

- 所有被改答案的题，速查表必须同步更新
- 所有被重写题面的题，解析必须同步改写
- 错项分析不能停留在“其他项错误”这种弱解释
- 要明确指出错项错在什么边界

## 质量标准

本次重做完成后，应满足以下标准：

### 1. 题目不能靠常识秒杀

理想状态不是“读题很难”，而是“读完题必须做判断”。因此：

- 不能再出现明显离题选项
- 不能再出现只看语气就知道哪项更像正确答案的情况

### 2. 单选答案分布自然

完成后复查单选答案分布：

- 不追求机械平均
- 但必须避免显著集中在单一字母

### 3. 多选答案组合不再机械重复

完成后复查多选答案组合：

- 不允许继续出现大面积相同组合
- 不允许让“猜 `ABC`”成为高成功率策略

### 4. 解析支持复盘

每道题的解析都应至少回答：

- 正确项为什么对
- 最关键的错项为什么错
- 这题考查的是哪个知识点边界

## 验收标准

完成后应满足以下条件：

- `C3` 保持 `26` 题规模不变
- 题号 `1-26` 保持不变
- 单选题答案不再明显集中在 `B`
- 多选题答案组合不再大面积集中在 `ABC`
- 至少一半以上题目的错误选项具备同维度竞争性
- 被改动题目的解析全部同步更新
- 做题者不能主要依靠答案分布规律或常识排除拿高分

## 非目标

本次不做以下内容：

- 不把 `C3` 改成场景题混合卷
- 不扩题到更大规模
- 不修改 `C2` 或 `C4`
- 不重写 `C3/考试大纲.md`

## 风险与控制

### 风险 1：为了打散答案分布而强行改答案

控制方式：

- 先按题面逻辑决定答案，再检查分布
- 不能为了分布好看而牺牲题目合理性

### 风险 2：为了提高难度而把题写偏

控制方式：

- 所有重写都必须落在现有课程与考点范围内
- 难度来自边界和条件，不来自冷门知识

### 风险 3：题面改了，但解析没跟上

控制方式：

- 按“题面 -> 速查表 -> 逐题解析”顺序同步改
- 最后做题号和答案一致性核对

## 后续路径

如果本轮 `C3` 中修重做达到预期，可把同一套标准继续用于：

- `C4` 的干扰项强化
- 已扩题模块的答案分布审计
- 后续整卷模拟卷抽题时的题目筛选标准
