# C2_构造问答系统模拟题 - 客观题版

## 说明

- 贴近真实 ACP 考试形式，仅保留单选题与多选题。
- 本练习版共 13 题，适合闭卷自测。
- 建议先独立作答，再对照客观题答案版复盘。

## 作答建议

- 单选题：先排除明显错误项，再比较剩余选项。
- 多选题：重点留意是否在考查完整流程、优化方法集合或评测方法体系。
- 若遇到 RAG 相关题目，优先从“检索链路”和“生成链路”两个层面拆解判断。
